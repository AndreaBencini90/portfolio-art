# portfolio-art
portfolio personal site

# 1. Spostati sul branch di destinazione (main)
git checkout main

# 2. (Consigliato) Aggiorna il main locale con quello remoto per evitare conflitti
git pull origin main

# 3. Esegui il merge del branch di sviluppo dentro al main
# (Sostituisci 'develop' con il nome esatto del tuo branch se diverso, es. 'dvelop')
git merge develop

# 4. Invia le modifiche aggiornate al repository remoto
git push origin main
