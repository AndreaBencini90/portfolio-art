Quantum graphity

The Spin Network with fixed variance represents a space that is not yet space: an essential network in which every node has the same number of connections. Before the universe takes shape, everything exists in a condition of perfect equanimity.

It is a structure without hierarchies, where nothing prevails and everything exists only through relation.
